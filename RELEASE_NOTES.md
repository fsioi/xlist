# XList WebDAV 文件管理器 - 发布说明

## 应用概述
XList 是一款功能强大的 WebDAV 文件管理器，允许用户通过 WebDAV 协议访问和管理远程服务器上的文件。

## 主要功能

### 1. WebDAV 文件管理
- 浏览远程服务器上的文件和文件夹
- 支持目录导航和文件预览
- 实时文件列表更新

### 2. 服务器管理
- 添加和管理多个 WebDAV 服务器
- 支持服务器认证（用户名/密码）
- 自动保存服务器配置

### 3. 文件操作
- 文件下载
- 添加到收藏夹
- 添加到最近访问

### 4. 用户界面
- 支持网格和列表两种视图模式
- 响应式设计，适配不同屏幕尺寸
- 加载状态和错误处理

## 如何使用

### 1. 添加服务器
1. 打开应用
2. 点击右上角的设置图标
3. 点击"服务器设置"
4. 点击"添加服务器"
5. 输入服务器 URL、用户名和密码
6. 点击"保存"

### 2. 浏览文件
1. 从服务器列表中选择一个服务器
2. 应用会自动连接到服务器并显示根目录的文件
3. 点击文件夹进入子目录
4. 点击文件预览或下载

### 3. 管理文件
1. 长按文件或文件夹显示操作菜单
2. 选择要执行的操作（下载、添加到收藏夹等）

## 已知问题

### 1. 构建问题
- 使用 NDK r27d 构建时可能会遇到 Clang 编译器找不到的问题
- 解决方案：确保设置了正确的 ANDROID_NDK_HOME 环境变量

### 2. 功能限制
- 目前不支持文件上传
- 不支持文件重命名和删除
- 不支持创建新文件夹

### 3. 性能问题
- 大文件列表加载可能较慢
- 网络连接不稳定时可能会出现超时
